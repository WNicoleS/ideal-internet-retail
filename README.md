## Usage

In '.env.EXAMPLE' file, remove '.EXAMPLE' and add SQL user and password.
In the terminal:

- run 'npm install'
- run 'npm i sql'
- run 'npm i sequelize'
- run 'npm i express'
- run 'npm i dotenv'
- Then, run 'npm start'

It will say that it is listening at port 3001.

- Go to Insomnia

From there, you can access either Categories, Products, or Tags
